# Used-Indian-Vehicle-Price-Prediction
This app is used to predict price of Indian Vehicle. Only Car or Bike

## Project Structure
```
├── app.py
├── train.py
├── data/
│ ├── indian_used_cars_raw_57480.csv
│ └── Used_Bikes.csv
├── LICENSE
└── README.md
```


## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Prath-Digital/Used-Indian-Vehicle-Price-Prediction.git
cd Used-Indian-Vehicle-Price-Prediction

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Train the model

```bash
python train.py
```

### 4. Run the app

```bash
python app.py
```
